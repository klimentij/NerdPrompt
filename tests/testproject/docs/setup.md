# Setup Guide

This is a sample setup guide for the test project.

## Prerequisites

- Python 3.9+
- pip

## Installation

Install the package with:

```bash
pip install -e .
```

## Configuration

Create a `.env` file with:

```
DEBUG=true
API_KEY=your_api_key_here
```

## Running

Run the main script:

```bash
python -m src.main
```
